# spm-case
some package which submit on spm3.6 .

## Prepare

````bash
$ npm install spm -g
````

Make sure your spm version is 3.6.x .

## Usage

````bash
# !-- 1. enter directory
$ cd case01-filesaver

# !-- 2. debug
$ spm server
$ open http://127.0.0.1:8000/

# !-- 3. build
$ spm build
$ open dist/index.html
````
